После нажатия кнопки _Load selected_ появляется надпись _Loading_. Нажатие на кнопку во время подключения игнорируется.
![image](/Pictures/MainWindow_Loading.png)

Особенности:
	- Из Mjpeg пакета парсится только картинка.
	- Сильно упрощённый париснг пакетов из стрима.
	- События инициализируются пустым обработчиком, чтобы не проверять на null

Недостатки:
	- Перевод картинки в стрим и обратно дважды, один раз в парсере, второй при отображении. Можно решить не парсив картинку, а оставить byte[].
	- Приложение запускается только после загрузки XML конфига, иногда приходится долго ждать.
	- ViewModel получилась достаточно сложной. Можно разделить ещё больше.
	- Сильная связанность между классами. Больше интерфесов
